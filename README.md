# sparse8b-phase0

CPU-only evaluation harness for TEAL-style activation sparsity on MLC-quantized Llama models
(Llama-3.1-8B q3f16_0 vs Llama-3.2-3B q4f16_0 baseline). Mirrored from a private research repo;
this public copy exists so the evaluation matrix can run on free GitHub Actions.

See phase0/README.md. Trigger runs via the Actions tab (workflow_dispatch).
